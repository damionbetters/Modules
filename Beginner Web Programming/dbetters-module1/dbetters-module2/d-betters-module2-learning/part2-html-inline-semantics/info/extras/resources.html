Resources

If linking to an external website, you can create the link by including the whole url as the href value. You can point to a specific page on a website by the use of the entire url of that page (i.e. https://example.com).

If linking to an internal webpage, you can use relative urls. a relative url uses the local files of a site used in the folder of the website. The syntax for relative url is /about/about.html for files in the same folder/directory. Or use ../about/about.html to access a parent folder and/or file.

P.S. Href stands for HyperText Reference. HTTP stands for HyperText Transport Protocol. HTTPS stands for HyperText Transport Protocol Secure.
